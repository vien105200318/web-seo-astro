# Character Animation

## The Problem

Spirit model đẹp. Nhưng di chuyển như... robot.

## Solution: Mixamo + Custom

- Base animations từ Mixamo (free!)
- Modify trong Blender để eerie hơn:
  - Slow down 50%
  - Add สี่ Jitter/shake
  - Unnatural joint bends
  - Float invece of walk

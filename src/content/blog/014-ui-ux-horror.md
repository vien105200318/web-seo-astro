# UI/UX Cho Horror Game

Horror cần immersion. UI quá nhiều = breaks atmosphere.

## Solution: Diegetic UI

- Không có health bar trên screen
- Inventory = physical bag player mở ra
- Objectives = nhật ký trong túi
- Interact prompts = subtle, fade nhanh

Inspiration từ Dead Space. Minimal but clear.

**UI Designer's nightmare, player's dream.**
